# Button

## Usage

```jsx live
<Button
  label='Button'
  variant='contained'
  color='pink'
  colorNum='300'
  size='base'
  radius='md'
  onClick={() => alert('Hey there!')}
/>
```

---

## Variants

Button variant can be access by using the `variant` attribute

### Contained

```jsx live
<Button
  label='Button'
  variant='contained'
  color='pink'
  colorNum='300'
  size='base'
  radius='md'
/>
```

---

### Ghost

```jsx live
<Button
  label='Button'
  variant='ghost'
  color='pink'
  colorNum='300'
  size='base'
  radius='md'
/>
```

---

### Text

```jsx live
<Button
  label='Button'
  variant='text'
  color='pink'
  colorNum='300'
  size='base'
  radius='md'
/>
```
